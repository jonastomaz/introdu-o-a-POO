Este repositório contém exemplos práticos de Programação Orientada a Objetos (POO) aplicados em diferentes domínios, 
com o objetivo de reforçar conceitos como classes, objetos, encapsulamento, herança e polimorfismo.

Os projetos foram desenvolvidos de forma didática para simular sistemas reais simples.

Projetos incluídos
Hub de Entregas

Sistema que simula a gestão de entregas, incluindo:

Cadastro de entregadores
Registro de pedidos
Acompanhamento de status de entrega
Atribuição de entregas a entregadores

Conceitos POO aplicados:

Classes e objetos
Relacionamento entre classes

Agendamento Clínico

Sistema de agendamento de consultas médicas.

Funcionalidades:

Cadastro de pacientes
Cadastro de médicos
Agendamento de consultas

Conceitos POO aplicados:

Abstração
Associações entre classes

Cadastro Acadêmico

Sistema para gerenciamento de informações acadêmicas.

Funcionalidades:

Cadastro de alunos
Cadastro de disciplinas
Matrícula em disciplinas

Conceitos POO aplicados:

Organização de entidades

Chá de Bebê

Sistema simples para organização de eventos.

Funcionalidades:

Lista de convidados
Controle de presentes

Conceitos POO aplicados:

Modelagem de eventos
Encapsulamento
Manipulação de listas de objetos

Sistema Bancário

Simulação de operações bancárias básicas.

Funcionalidades:

Abertura de contas
Depósito e saque
Transferência entre contas
Consulta de saldo

Conceitos POO aplicados:

Regras de negócio

O objetivo principal é praticar e demonstrar conceitos fundamentais de POO
através de pequenos sistemas simulados, facilitando o aprendizado progressivo da modelagem orientada a objetos usando a linguagem java.
